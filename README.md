# 南京工业大学BBS论坛

### 项目简介
> 本项目基于 [renren-fast](https://cloud.tencent.com/developer/article/1165985)进行二次开发，为避免误解已修改部分类名

该项目旨在为学校搭建一个BBS论坛
非商业用途，仅供学习交流

#### 技术选型
* 后端
    * spring boot
    * mybits plus
    * shiro
    * swagger2
    * redis
    * mysql
    * quartz
* 前端
    * vue
    * element-ui
